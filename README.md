# Ridesharing Android App using Java and XML with Firebase

WeRide is a peer to peer(P2P) ridesharing mobile application that enables users to share their rides with other commuters from the same organisation(i.e. same college/university/workplace) travelling along the same route. The app aims to provide a platform for users to connect with each other and share their journey, reducing traffic congestion, air pollution, and travel expenses. P2P carpooling apps can provide an alternative to traditional ride-hailing services and public transportation, especially in areas with limited public transportation options.

Users can sign up on the app as a driver or a passenger and create their profiles with details such as their name, contact information(workplace email-id) and route details.When a user creates a ride, the app matches the driver with potential passengers who are travelling along the same route. Passengers can request a ride with the driver. Payemnt has to be done using Razorpay to confirm the ride.
